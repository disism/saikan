# disism saikan 


## Command Line

### OpenID Connect
#### Add

Add the OpenID Connect Provider supported by the current application.

```bash
saikan oidc add -n disism -c http://localhost:3033/.well-known/openid-configuration
```

#### List

```bash
saikan oidc list
```

#### Remove

```bash
saikan oidc remove -id [ID]
```
